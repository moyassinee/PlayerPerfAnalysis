# Merge stats and keep most recent metadata
df = shooting_df.merge(passing_df, on=['player_', 'season_', 'team_'], how='outer')
# ... additional merges for defense/possession ...

most_recent = (
    df.sort_values("season_", ascending=False)
    .groupby("player_")[["team_", "age_", "league_", "pos_"]]
    .first().reset_index()
)

df_agg = (
    df.groupby("player_")[cumulative_cols]
    .sum(min_count=1).reset_index()
    .merge(most_recent, on="player_", how="left")
)

Design Decisions

KEY CHOICES

📊
Cumulative Aggregation
Sums stats across multiple seasons/teams to get a wider sample size of player behavior.
⏱️
Volume Threshold
The 20-90s filter ensures we aren't comparing Salah to "one-season wonders" or small samples.
📐
Selective Normalization
Smart logic divides only cumulative totals by 90s, preserving existing per-90 metrics.
🧬
Metadata Preservation
Uses .first() on sorted seasons to ensure Team and League info is current.
